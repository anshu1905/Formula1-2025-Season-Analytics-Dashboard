# Formula1-2025-Season-Analytics-Dashboard

A sleek, data driven analytics dashboard for the Formula 1 2025 season delivering deep insights into driver performance, team dominance, race trends and season long comparisons Built for speed clarity and serious F1 nerds because vibes are good but lap time deltas are better.

## Steps to run the application/dashboard:

1. Run the ```requirements.txt``` file first to download all the required libraries.
2. Run the ```app.py``` application/dashboard.
